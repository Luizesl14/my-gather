# Indice De Origem

## Sprites

- `images/trae/<personagem>/.../<variant>` -> `assets/sprites/characters/normal/...`
- `images/trae/<personagem>/.../<variant-combat>` -> `assets/sprites/characters/combat/...`
- `images/trae/hair/<gender>/<pack>` -> `assets/sprites/customization/hair/<gender>/<pack>`
- `images/trae/actions/*bubble*.png` -> `assets/sprites/bubbles/source/`
- `images/trae/actions/*gesture*.png` -> `assets/sprites/gestures/source/`

## Environment

- `images/trae/components/office/*` -> `assets/environment/themes/office/*`
- `images/trae/components/industrial/*` -> `assets/environment/themes/industrial/*`

## Maps

- `images/trae/maps/*.png` -> `assets/maps/models/`
