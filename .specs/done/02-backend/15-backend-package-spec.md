# 15 — Backend Package Spec

## Objetivo

Criar projeto backend Node.js, TypeScript e Fastify.

## Criar

- `backend/package.json`
- `backend/tsconfig.json`
- `backend/src/main.ts`
- `backend/src/app.ts`

## Dependências base

- `fastify`
- `typescript`
- `tsx` ou equivalente para dev
- `zod`
- `dotenv`

## Scripts

- `dev`
- `build`
- `start`
- `test`

## Critérios de aceite

- `npm run build` compila.
- `GET /health` responde.
