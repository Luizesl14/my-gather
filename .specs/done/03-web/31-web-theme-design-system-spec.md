# 31 — Web Theme & Design System Spec

## Objetivo

Implementar tema Gather-like light/dark no Flutter.

## Criar

- `app_theme.dart`
- `app_colors.dart`
- `app_spacing.dart`
- `app_radius.dart`
- `app_typography.dart`
- componentes base em `shared/design_system/`

## Componentes base

- Botão icon-only.
- Tooltip.
- Panel.
- Modal.
- Badge.
- Status dot.

## Critérios de aceite

- App alterna light/dark.
- Tokens seguem `02-theme-layout-spec.md`.
- Sem cores hardcoded fora do tema.
