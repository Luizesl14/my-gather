# 40 — Toolbar Actions Spec

## Objetivo

Criar toolbar inferior de ações rápidas.

## Ações MVP

- Chat.
- Voz/chamada.
- Chamar usuário.
- Acenar.
- Café.
- Reunião.
- Ajuda.
- Gritar.
- Notificações.
- Mais ações.

## Estados

- Default.
- Hover.
- Pressed.
- Active.
- Disabled.
- Badge.

## Critérios de aceite

- Botões têm tooltip.
- Toolbar não cobre avatar.
- Light/dark funcionam.
