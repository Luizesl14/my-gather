# 44 — Bubble Animation Spec

## Objetivo

Detalhar balões acima de avatar, mesa ou objeto.

## Tipos

- Typing.
- Speech.
- Voice.
- Call.
- Invite.
- Knock.
- Bell.
- Question.
- Alert.
- Shout.
- Coffee.
- Help.
- Reaction.

## Prioridade

1. Call/alert.
2. Help/shout.
3. Invite/coffee/knock.
4. Reaction/speech.
5. Typing.

## Critérios de aceite

- Balões não sobrepõem nome.
- Typing expira.
- Chamada substitui reação.
