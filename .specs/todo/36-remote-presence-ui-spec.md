# 36 — Remote Presence UI Spec

## Objetivo

Renderizar outros usuários e presença realtime.

## Elementos

- Avatar remoto.
- Nome.
- Indicador de status.
- Interpolação de movimento.
- Entrada/saída visual.

## Estados de presença

- Disponível.
- Ausente.
- Ocupado.
- Em reunião.
- Foco.
- Offline.

## Critérios de aceite

- Movimento remoto é suave.
- Status muda sem recriar avatar.
- Usuário que sai desaparece com fade curto.
