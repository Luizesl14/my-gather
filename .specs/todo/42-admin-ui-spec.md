# 42 — Admin UI Spec

## Objetivo

Criar interface admin MVP.

## Telas

- Membros.
- Mesas.
- Salas.
- Publicar mapa JSON.

## Ações

- Listar membros.
- Atribuir mesa.
- Criar sala.
- Editar sala.
- Validar mapa.
- Publicar mapa.

## Critérios de aceite

- Apenas admin/owner acessa.
- Validação bloqueia mapa inválido.
- Mudanças refletem no workspace.
