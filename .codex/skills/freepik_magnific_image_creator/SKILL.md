# Freepik Magnific Image Creator

## Prioridade

Use toda a capacidade de planejamento antes da chamada paga. Creditos sao gastos somente quando o prompt, parametros e criterio de aceite estiverem claros.

## Antes de Chamar o MCP

1. Confirme o objetivo da imagem e o destino: sprite, cenario, marketing, mockup, textura, retrato, UI asset.
2. Verifique se ja existe asset/referencia local reutilizavel.
3. Defina tamanho, proporcao, transparencia, estilo e qualidade alvo.
4. Escreva prompt positivo e negativo.
5. Defina criterio objetivo de sucesso.
6. Se a ferramenta permitir, rode primeiro preview/low-cost; so faca upscale/enhance depois de aprovar base.

## Politica de Creditos

- Nao faca exploracao aleatoria.
- Limite variacoes iniciais a 1 ou 2 quando o briefing for bom.
- Use upscale/enhance apenas em imagem aprovada.
- Para sprites, gere folha pequena ou personagem isolado antes de muitos frames.
- Para cenarios, aprove composicao em baixa antes de pedir detalhe alto.
- Quando uma tentativa falhar, explique a causa provavel e altere somente os pontos necessarios.

## Prompt Freepik/Magnific

```text
Create [asset type] for [project/use].
Main subject: [clear subject].
Composition: [view, framing, pose, depth].
Style: [specific visual style].
Rendering: crisp details, clean shapes, production-ready asset.
Color/lighting: [palette and lighting].
Technical constraints: [aspect ratio], [transparent background if needed], no text, no watermark, no logo.
Avoid: blurry, noisy, distorted, extra objects, inconsistent perspective, cropped subject.
```

## Upscale/Enhance

Use upscale quando:

- A composicao ja esta correta.
- O estilo esta aprovado.
- O asset precisa de mais nitidez para producao.

Nao use upscale quando:

- O personagem/objeto esta errado.
- A pose ou camera nao serve.
- O fundo precisa mudar.
- Ha artefatos estruturais graves.

## Saida Esperada

Ao finalizar, informe:

- prompt usado;
- parametros relevantes;
- custo/numero de chamadas, se disponivel;
- arquivo final gerado;
- proximos ajustes recomendados sem gastar credito desnecessario.
