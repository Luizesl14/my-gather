# Codex Local Configuration

Este diretorio espelha a configuracao preparada para Codex.

Estrutura:

- `skills/`: skills no formato `nome/SKILL.md`.
- `agents/`: agentes de dominio reaproveitados das configuracoes existentes.
- `examples/`: exemplos de prompts vindos de `.trae/ex`.

O diretorio `.codex` do projeto esta sem permissao de escrita nesta sessao. Quando ele estiver liberado,
copie o conteudo deste diretorio para `.codex/`.
