# Codex Project Instructions

Este projeto usa instrucoes especializadas em `.codex-project/skills`.

Antes de tarefas tecnicas, verifique se alguma skill local se aplica ao pedido:

- `typescript_specialist`: TypeScript avancado, tipagem, generics, narrowing e APIs type-safe.
- `flutter_especialist`: Flutter, Dart, widgets, estado, navegacao, performance e testes.
- `devops_especialist`: CI/CD, containers, deploy, observabilidade e infraestrutura.
- `domain_drive_desing`: DDD, bounded contexts, agregados, eventos e linguagem ubiqua.
- `engenheiro_especialist`: arquitetura, qualidade, revisao tecnica e engenharia de software.
- `java_especialist`: Java, JVM, concorrencia, APIs, testes e refactors.
- `kotlin_especialist`: Kotlin, coroutines, DSLs, multiplatform e interoperabilidade.
- `node_especialist`: Node.js, APIs, backend, runtime, pacotes e performance.
- `spec_drive_development_especialist`: desenvolvimento orientado por especificacao.
- `spring_especialist`: Spring Boot, Spring Data, seguranca, configuracao e testes.
- `character`: criacao e consistencia de personagens.

Para escritorio virtual inspirado no Gather, use `.codex-project/agents/gather_virtual_office_specialist.md`
como referencia de dominio.

Se o diretorio `.codex` estiver gravavel, mantenha as mesmas skills em `.codex/skills`.
